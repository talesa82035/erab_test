■各機能の説明

・覗き見システム
　対象を覗き見て、「おっ！やってんねえ！＾＾」を愉しむシステムです。
　解放には「魔法知識」がLv1以上で発生するサブクエスト「使い魔の創造」のクリアが必要です。
　
　
・性癖：ゆきずりセックス
　性欲がたまるとその辺の観光客(モブ)を誘惑して物陰でヤり始めます。
　野外オナニーの亜種となります。
　今はまだ女性キャラしか対応してません。
　
　現場で覗く、もしくは使い魔を通じて状況描写があります。
　一定の確率で覗きバレが発生し、以後見せつけセックスとなります。

　独自の立ち絵変更処理を持っており、
　　短冊_XXX_差分_覗きセックス
　　短冊_XXX_差分_覗きＡセックス
　　短冊_XXX_差分_覗き奉仕
　　短冊_XXX_差分_覗きバレセックス
　　短冊_XXX_差分_覗きバレＡセックス
　　短冊_XXX_差分_覗きバレ奉仕
　のファイルがあればそれが表示されます。
